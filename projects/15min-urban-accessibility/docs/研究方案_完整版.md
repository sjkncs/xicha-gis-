# 15分钟城市陷阱：公共服务时空可达性幻觉下的时间贫困地图

## 研究方案 v2.0 | 2026-05-19 | 深圳市南山区

---

## 一、研究背景与问题

### 1.1 "15分钟城市"的概念与局限

"15分钟城市"（15-minute city）由法国城市学家Carlos Moreno提出，强调居民在15分钟步行或骑行范围内可获得日常生活所需的基本服务。这一理念在新冠疫情后被全球许多城市采纳为城市更新和社区复兴的政策工具。

然而，现有"15分钟生活圈"规划和评价存在以下核心局限：

| 传统评价维度 | 忽略的隐性因素 | 实际影响 |
|------------|---------------|---------|
| 直线距离 | 巷道曲折系数 | 城中村实际步行距离可能是直线距离的1.5-2倍 |
| 设施数量 | 排队等待时间 | 高峰时段服务等待时间可超过30分钟 |
| 空间分布 | 开放时间错配 | 夜间开放的便利店比例可能不足20% |
| 服务半径 | 心理威慑门槛 | 手续复杂、语言障碍等认知成本 |
| 工作日可达 | 夜间服务缺失 | 下班后无法获得基本服务 |

### 1.2 研究问题

**核心问题**：表面上满足"15分钟距离标准"的社区，居民实际获取公共服务是否仍然耗费更长时间和更高精力？

**子问题**：
1. 不同社区类型（城中村vs高端社区）的实际服务获取时间差异有多大？
2. 夜间时空可达性相比白天如何恶化？
3. 时间贫困指数能否揭示传统空间可达性地图中的"服务盲区"？

## 二、理论框架

### 2.1 时间贫困（Time Poverty）理论

**概念定义**：时间贫困是指个体可用于自由支配的时间低于社会可接受最低标准的状态。与收入贫困类似，时间贫困也是一种能力剥夺——它限制了人们参与社会、经济和文化活动的机会。

**时间贫困指数（Time Poverty Index, TPI）公式**：

$$TPI = \frac{T_{commute} + T_{wait} + T_{service}}{T_{available}}$$

其中：
- $T_{commute}$：实际通勤时间（含曲折系数修正）
- $T_{wait}$：平均等待时间
- $T_{service}$：标准办理时间
- $T_{available}$：单次出行的可支配时间预算（假设为120分钟）

### 2.2 有效服务时间模型

**定义**：从出发到完成服务并返回的总时间消耗

$$T_{effective} = T_{travel} \times F_{detour} + T_{wait} + T_{service}$$

#### 2.2.1 移动时间（$T_{travel}$）

使用OSMNX进行网络分析，计算从小区质心到最近服务设施的最短路径时间：

$$T_{travel} = \frac{L_{shortest}}{v_{walk}}$$

其中 $v_{walk}$ 为步行速度（假设为5km/h）。

#### 2.2.2 曲折系数（$F_{detour}$）

反映巷道曲折、障碍物等对实际行程的偏离：

$$F_{detour} = 1 + \alpha \times \left(1 - \frac{D_{road\_density}}{D_{max}}\right)$$

| 社区类型 | 曲折系数 $\alpha$ | 说明 |
|--------|-----------------|------|
| 高端社区 | 1.0 | 规整路网，直接可达 |
| 普通商品房 | 1.1 | 封闭小区，略有绕行 |
| 老旧小区 | 1.3 | 设施老化，动线混乱 |
| 城中村 | 1.5 | 巷道曲折，缺乏规划 |

#### 2.2.3 等待时间（$T_{wait}$）

基于设施类型和高峰时段估算：

$$T_{wait} = \begin{cases} 
0 & \text{预约制或无人排队} \\
t_{peak} \times \frac{D_{demand}}{C_{capacity}} & \text{现场办理}
\end{cases}$$

#### 2.2.4 办理时间（$T_{service}$）

| 设施类型 | 标准办理时间 | 说明 |
|---------|------------|------|
| 便利店购物 | 5分钟 | 快速服务 |
| 快递寄取 | 8分钟 | 含排队 |
| 药店购药 | 10分钟 | 咨询+购药 |
| 菜市场买菜 | 15分钟 | 选品+结账 |
| 医院挂号 | 30分钟 | 挂号+等候 |
| 银行办事 | 45分钟 | 取号+办理 |

### 2.3 时间窗口匹配度

**定义**：设施开放时间与居民可出行时间的重叠程度

$$M_{time} = \frac{T_{overlap}}{T_{demand}}$$

其中 $T_{overlap}$ 为设施开放时段与居民可出行时段的交集时长。

**时段划分**：

| 时段 | 时间范围 | 居民可出行性 |
|-----|---------|-------------|
| 白天标准 | 08:00-18:00 | 高（工作时间可请假） |
| 夜间下班 | 18:00-22:00 | 最高（下班后自由时间） |
| 深夜应急 | 22:00-08:00 | 低（仅应急需求） |

### 2.4 设施开放时间的三种公平观

| 公平观 | 目标 | 政策含义 |
|--------|------|---------|
| **功利主义** | 最大化整体可达性 | 延长营业时间 |
| **平等主义** | 保证人人平等获取 | 均匀分布开放时段 |
| **分配主义** | 优先保障弱势群体 | 针对性补缺 |

### 4.3 社区类型识别

基于建筑数据识别社区类型：

| 特征指标 | 城中村 | 老旧小区 | 普通商品房 | 高端社区 |
|---------|-------|---------|-----------|---------|

### 4.4 时间可达性计算

#### 4.4.1 白天时段（08:00-18:00）

所有设施按正常开放时间计算。

#### 4.4.2 夜间时段（18:00-22:00）

过滤条件：
- 仅保留开放时间覆盖18:00-22:00的设施
- 便利店标记夜间服务字段 = 1
- 医院标记24小时急诊

#### 4.4.3 时间贫困指数计算

**分项指数**：
1. **移动时间指数（MTI）**
   $$MTI = \frac{T_{travel} \times F_{detour}}{T_{threshold}}$$

2. **等待时间指数（WTI）**
   $$WTI = \frac{T_{wait}}{T_{threshold}}$$

3. **窗口匹配指数（TMI）**
   $$TMI = 1 - M_{time}$$

**综合时间贫困指数（CTPI）**：
$$CTPI = w_1 \times MTI + w_2 \times WTI + w_3 \times TMI$$

权重由层次分析法（AHP）确定，或等权重设为1/3。
