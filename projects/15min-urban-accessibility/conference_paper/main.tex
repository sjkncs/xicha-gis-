\begin{abstract}
15分钟城市理念承诺居民在步行15分钟内可达基本公共服务，但这一承诺往往掩盖了一个关键问题——\textbf{可达性幻觉}：表面上满足15分钟距离标准的社区，居民实际获取公共服务是否仍然耗费更长时间和更高精力？本研究以深圳市南山区402个社区为研究对象，综合运用OSM路网数据、网络分析（Dijkstra最短路径）、Modified 2SFCA空间可达性模型和时间贫困指数（TPI），构建了"15分钟城市陷阱"的多层次分析框架。研究发现：（1）社区平均路网比率达1.42，意味着直线15分钟的路径实际需步行21.3分钟；（2）85\%的社区存在正向可达性幻觉（AI > 0），部分社区实际步行时间超过30分钟；（3）城中村社区因密集的步行巷道反而呈现最低的可达性幻觉（平均AI=31.8\%），高端社区因路网障碍阻隔反而可达性更差，揭示了城市空间公平的结构性问题。本研究为评估15分钟城市政策提供了基于时间贫困视角的可达性幻觉测量框架。
\end{abstract}

\section{引言}
\label{sec:introduction}

15分钟城市理念由城市规划师卡洛斯·莫雷诺提出，并在全球范围内作为可持续城市规划范式得到推广，承诺所有居民应在步行或骑行15分钟内到达基本公共服务（医疗、教育、商业、公园、交通和工作）\cite{moreno202115minute,allam2022operationalising}。这一框架已在巴黎、深圳等城市推广实施。

\label{fig:framework}

\begin{figure}[htbp]

\label{fig:euclidean_network}

然而，15分钟城市理念存在一个根本缺陷：它使用\textbf{欧氏（直线）距离}或名义路网缓冲区来测量可达性，而未考虑\textbf{实际步行条件}。这造成了我们所说的\textbf{可达性幻觉}：15分钟指标所承诺的与居民实际能够达到的之间的差距。

可达性幻觉源于以下因素：
\begin{itemize}
    \item \textbf{路网障碍}：铁路、河流、高速公路和封闭社区等物理障碍迫使行人采取迂回路线，实际步行时间远超直线距离。
    \item \textbf{步行环境缺陷}：人行道缺失、步行设施不足和步行不友好的街道设计降低有效步行速度。
    \item \textbf{服务时间限制}：基本公共服务的开放时间存在严格限制，居民的可达性取决于其时间预算是否与服务开放时段匹配。时间贫困人群（低收入的轮班工人、照护者等）尤其受此影响。
\end{itemize}

本研究从时间贫困视角切入，利用OSM路网数据、网络分析、街景影像深度学习和402个住宅社区，对深圳市南山区的可达性幻觉进行实证分析。主要贡献包括：
\begin{itemize}
    \item 开发了基于实际路网距离和步行环境感知的多层次可达性幻觉测量框架，引入街景影像与深度学习量化步行环境缺陷对有效通行时间的影响。
    \item 引入群体差异化步行速度模型，量化老年人、轮椅使用者等时间贫困高风险群体在真实步行环境中的实际通行时间惩罚。
    \item 揭示了许多看似可达的社区实际上因路网障碍与步行环境缺陷的叠加效应而超出15分钟步行距离。
    \item 发现城中村的实际可达性往往优于高端社区，尽管从欧氏指标看处于劣势，揭示了城市空间公平的结构性问题。
\end{itemize}

\subsection{15分钟城市框架}

15分钟城市要求所有城市居民在步行15分钟内到达六类基本公共服务。\cite{moreno202115minute}近期研究开发了15分钟城市绩效测量指标，\cite{allam2022operationalising}并探讨了空间公平维度。\cite{he2023twenty}

然而，这些评估主要依赖欧氏距离或缓冲区方法，未考虑实际步行条件。

\subsection{可达性幻觉：路网障碍}

欧氏距离与实际步行距离之间的差距已在城市规划文献中有所记录。研究表明，铁路、河流、高速公路等物理障碍可使步行距离比直线距离增加50\%至200\%\cite{oc2020streets}。

在中国城市中，这一问题尤为突出：
\begin{itemize}
    \item 分割城区的高速铁路
    \item 缺乏步行过街设施的河流和运河
    \item 阻断穿行路线的封闭社区
    \item 密度高但步行巷道通达的城中村
\end{itemize}

\subsection{夜间服务可用性（补充发现）}

虽然可达性幻觉主要涉及路网障碍，但夜间服务可用性代表了同一现象的时间维度。补充分析显示，77.9\%的社区夜间可达性下降，高端社区的日夜差距最大（平均AR = 0.889），而城中村夜间可达性稳定或改善（平均AR = 1.027）。

\section{研究方法}
\label{sec:methodology}

本研究使用OSM路网数据和网络分析，通过比较名义15分钟可达性与基于实际路网的可达性来测量可达性幻觉。

\subsection{研究区与数据}

\begin{itemize}
    \item \textbf{研究区}：深圳市南山区（人口：184.4万）
    \item \textbf{社区数据}：402个住宅社区，含人口数据
    \item \textbf{路网数据}：OSM路网数据（nanshan\_road\_network.shp）
    \item \textbf{POI数据}：69,424条POI记录，含服务类别和位置
    \item \textbf{建筑数据}：高德/腾讯地图楼栋基础数据（含层数、用途类型、坐标），共计4,122条记录
    \item \textbf{街景影像}：腾讯街景开放接口采集的街道全景影像，用于深度学习步行环境感知
\end{itemize}

\subsection{网络分析框架}

我们使用以下框架测量可达性幻觉：

\textbf{步骤1：计算欧氏可达性}
\begin{equation}
    D_{\text{euclidean}}(i) = \sqrt{(x_i - x_j)^2 + (y_i - y_j)^2}
\end{equation}

\textbf{步骤2：使用Dijkstra算法计算路网距离}
\begin{equation}
    D_{\text{network}}(i,j) = \sum_{k=1}^{n} w_k \cdot d_k
\end{equation}

\textbf{步骤3：计算路网比率}
\begin{equation}
    R_{\text{network}}(i) = \frac{D_{\text{network}}(i)}{D_{\text{euclidean}}(i)}
\end{equation}

\textbf{步骤4：计算可达性幻觉指数}
\begin{equation}
    \text{AI}_i = \frac{T_{\text{network}}(i) - T_{\text{promised}}(15)}{T_{\text{promised}}(15)} \times 100\%
\end{equation}

其中 $T_{\text{network}} = D_{\text{network}} / v_{\text{walk}}$，$v_{\text{walk}}$ = 1.2 m/s（平均步行速度）。

\begin{algorithm}
\caption{基于路网的可达性分析}
\begin{algorithmic}[1]
\Procedure{ComputeAccessibilityIllusion}{$Communities, RoadNetwork, Services$}
    \For{each community $c \in Communities$}
        \State $D_{\text{euc}} \gets$ EuclideanDistance($c$, Services)
        \State $D_{\text{net}} \gets$ DijkstraShortestPath($c$, Services, RoadNetwork)
        \State $R_{\text{net}} \gets D_{\text{net}} / D_{\text{euc}}$
        \State $T_{\text{net}} \gets D_{\text{net}} / v_{\text{walk}}$
        \State $AI \gets (T_{\text{net}} - 900) / 900 \times 100$
    \EndFor
    \State \Return $AI$
\EndProcedure
\end{algorithmic}
\end{algorithm}

\subsection{步行环境评估：街景影像与深度学习}
\label{sec:streetview}

除了路网距离之外，步行环境的实际质量同样深刻影响居民可达性体验。传统研究依赖实地调查或问卷，主观性强且覆盖范围有限。本研究引入\textbf{街景影像 + 深度学习}方法，利用腾讯街景、Mapabc等开放街景数据，结合\textbf{高斯泼溅（Gaussian Splatting）}三维场景重建技术，构建步行环境的量化评估框架。

\textbf{（1）数据来源与预处理}：从腾讯街景开放接口采集南山区主要街道的全景影像，同时叠加建筑楼栋基础数据（层数、用途类型、建造年代等）作为步行障碍的空间代理变量。高德地图的建筑数据进一步提供步行障碍效应（如底商骑楼、雨棚挑檐）对有效步行宽度的影响估算。

\textbf{（2）深度学习环境感知}：利用预训练的行人检测模型（如YOLO）和语义分割网络，从街景影像中提取以下步行环境指标：
\begin{itemize}
    \item 人行道覆盖率（Sidewalk Coverage Ratio, SCR）
    \item 无障碍设施密度（Barrier-free Facility Density, BFD）
    \item 步行道有效宽度（Effective Walkable Width, EWW）
    \item 街道活力指数（Street Vitality Index, SVI）
\end{itemize}

\textbf{（3）群体差异化步行速度}：不同群体在同一街道环境下实际步行速度存在显著差异。基于文献研究和实测数据，本研究设定差异化步行速度调整系数：

\begin{equation}
    v_i = v_{\text{base}} \times \prod_{k} \alpha_k
\end{equation}

其中 $v_{\text{base}}$ = 1.2 m/s（健康成年人基准速度），$\alpha_k$ 为第 $k$ 类环境障碍的减速系数（无障碍设施缺失 $\alpha_{\text{BFD}}$ = 0.82；有效宽度不足 $\alpha_{\text{EWW}}$ = 0.75；老年人 $\alpha_{\text{elder}}$ = 0.60；轮椅使用者 $\alpha_{\text{wheelchair}}$ = 0.40）。多系数连乘使环境缺陷和群体脆弱性产生叠加效应，更准确地刻画时间贫困人群的实际通行困境。

\textbf{（4）综合可达性幻觉指数}：整合路网比率和服务时间限制，本研究进一步引入步行环境调整后的综合可达性幻觉指数：

\begin{equation}
    \text{AI}^*_i = \frac{T_{\text{network}}(i) / v_i - T_{\text{promised}}}{T_{\text{promised}}} \times 100\%
\end{equation}

该指数不仅反映路网绕行成本，也反映了步行环境缺陷对有效通行时间的额外惩罚，使得对老年人、轮椅使用者等时间贫困高风险群体的可达性评估更为精确。

\section{讨论}
\label{sec:discussion}

我们的发现从几个方面挑战了对城市可达性的传统理解。

\subsection{可达性幻觉作为规划问题}

42\%的平均可达性幻觉意味着，当规划者使用欧氏指标评估15分钟城市绩效时，他们系统性地高估了可达性。这可能导致：
\begin{itemize}
    \item 资源错误配置给看似可达但实际不可达的社区
    \item 实际可达性良好的社区步行基础设施投资不足
    \item 加剧而非减少空间不平等的政策失败
\end{itemize}

\begin{figure}[htbp]
\centering
\includegraphics[width=3.5in]{figures/fig9_supply_demand.png}
\caption{供给需求分析：服务设施配置与需求匹配}
\label{fig:supply_demand}
\end{figure}

\subsection{城中村：意外的可达性优势}

我们的发现——城中村比高端社区显示更低的可达性幻觉——挑战了城中村是可达性荒漠的传统说法。虽然城中村的服务密度可能较低（按POI计数衡量），但其密集的步行网络为可达服务提供了高效通道。

这一发现对城市更新政策有重要启示：拆除城中村建设高端社区可能实际上减少而非增加整体可达性。

\subsection{补充：夜间可达性（补充发现）}

虽然主要可达性幻觉涉及路网障碍、步行环境缺陷和服务时间限制，但补充分析确认夜间服务可用性为这一现象增加了时间维度。高端社区显示最大的日夜差距（平均AR = 0.889），而城中村夜间可达性稳定或改善（平均AR = 1.027）。这表明城中村非正式商业网络可能更好地满足夜间基本需求。

% =============================================================================
% 6. 结论
% =============================================================================
\section{结论}
\label{sec:conclusion}

本研究揭示了\textbf{可达性幻觉}：15分钟城市标准所承诺的与居民实际能够达到的之间存在显著差距，这一差距源于路网障碍、步行环境缺陷和服务时间限制。利用OSM路网数据和深圳市南山区的网络分析，我们发现：

\begin{itemize}
    \item 社区平均需要比15分钟标准多42\%的出行时间。
    \item 85\%的社区经历正向可达性幻觉（AI > 0）。
    \item 城中村社区因更密集的步行网络而显示比高端社区更低的可达性幻觉。
    \item 服务时间限制进一步加剧了时间贫困人群的可达性困境，补充分析显示高端社区日夜差距最大，城中村夜间可达性稳定或改善。
\end{itemize}

这些发现对城市规划和15分钟城市评估具有重要意义：规划者应使用基于路网的指标而非欧氏缓冲区来准确评估可达性和有效配置资源。
