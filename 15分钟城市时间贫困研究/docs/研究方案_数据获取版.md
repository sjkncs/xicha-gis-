## 一、研究背景与问题

### 1.1 "15分钟城市"的概念与局限

"15分钟城市"（15-minute city）由法国城市学家Carlos Moreno提出，强调居民在15分钟步行或骑行范围内可获得日常生活所需的基本服务。然而，现有"15分钟生活圈"规划和评价存在以下核心局限：

| 传统评价维度 | 忽略的隐性因素 | 实际影响 |
|------------|---------------|---------|
| 直线距离 | 巷道曲折系数 | 城中村实际步行距离可能是直线距离的1.5-2倍 |
| 设施数量 | 排队等待时间 | 高峰时段服务等待时间可超过30分钟 |
| 空间分布 | 开放时间错配 | 夜间开放的便利店比例可能不足20% |
| 服务半径 | 心理威慑门槛 | 手续复杂、语言障碍等认知成本 |
| 工作日可达 | 夜间服务缺失 | 下班后无法获得基本服务 |

### 1.2 核心研究问题

**核心问题**：表面上满足"15分钟距离标准"的社区，居民实际获取公共服务是否仍然耗费更长时间和更高精力？

## 三、理论框架

### 3.1 时间贫困指数（Time Poverty Index）

$$TPI = \frac{T_{commute} + T_{wait} + T_{service}}{T_{available}}$$

### 3.2 有效服务时间模型

$$T_{effective} = T_{travel} \times F_{detour} + T_{wait} + T_{service}$$

其中：
- $T_{travel}$：网络分析最短路径时间
- $F_{detour}$：曲折系数（城中村1.5，高端社区1.0）
- $T_{wait}$：排队等待时间
- $T_{service}$：标准办理时间

## 五、设施营业时间数据采集

### 5.1 高德API字段

```python
# 高德POI返回的关键字段
{
    "name": "便利店名称",
    "location": "113.xxx,22.xxx",  # 经纬度
    "type": "便利超市",
    "biz_ext": {
        "business_period": "08:00-23:00",  # 营业时间
        "cost": "25"  # 人均消费
    }
}
```

### 5.2 时段划分

| 时段 | 时间范围 | 居民可出行性 | 重点关注 |
|-----|---------|-------------|---------|
| 白天标准 | 08:00-18:00 | 高 | 医疗、教育 |
| 夜间下班 | 18:00-22:00 | **最高** | **便利店、快递、餐饮** |
| 深夜应急 | 22:00-08:00 | 低 | 药店、便利店 |
