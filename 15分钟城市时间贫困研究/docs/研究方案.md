# 15分钟城市陷阱：公共服务时空可达性幻觉下的时间贫困研究
## 研究方案 v1.0 | 2026-05-19

---

## 一、研究问题与核心假设

### 1.1 核心研究问题
传统"15分钟生活圈"仅以**步行距离/时间**为评价标准，忽略了以下隐性时间成本：
1. **移动摩擦成本**：巷道曲折、障碍物导致的实际行程偏离
2. **排队等待成本**：高峰时段服务设施的排队时间
3. **时间窗口约束**：设施开放时间与居民可出行时间的错配
4. **心理威慑成本**：服务获取的认知门槛（如手续复杂、语言障碍）

### 1.2 核心假设
- **H1**：同一"距离可达"标准下，城中村居民的实际服务获取时间显著长于高端社区
- **H2**：夜间的时空可达性恶化幅度与社区社会经济地位呈负相关
- **H3**：时间贫困指数能够揭示传统空间可达性地图中的"服务盲区"

## 二、理论框架

### 2.1 时间贫困（Time Poverty）理论
**定义**：居民可用于自由支配的时间低于社会最低标准的状态

**时间贫困指数（TPI）构建：
$$TPI = \frac{T_{commute} + T_{wait} + T_{service}}{T_{available}}$$

其中：
- $T_{commute}$：实际通勤时间（含曲折系数）
- $T_{wait}$：平均等待时间
- $T_{service}$：办理时间
- $T_{available}$：可支配时间预算（假设为2小时/次出行）

### 2.2 时空棱镜模型（Space-Time Prism）
基于Hägerstrand时间地理学，构建个人在时空约束下的可达域：
- **时空预算**：个体可用于出行的时间窗口
- **棱镜边界**：物理可达与时间可达的交集
- **不可达区域**：即使距离满足但时间不满足的区域

### 2.3 设施开放时间的三种公平观
| 公平观 | 目标 | 政策含义 |
|--------|------|----------|
| 功利主义 | 最大化整体可达性 | 延长营业时间 |
| 平等主义 | 保证人人平等获取 | 均匀分布开放时段 |
| 分配主义 | 优先保障弱势群体 | 针对性补缺 |

## 四、研究方法

### 4.1 有效服务时间模型

**定义**：从出发到完成服务返回的总时间消耗

$$T_{effective} = T_{travel} \times F_{detour} + T_{wait} + T_{service}$$

其中：
- $T_{travel}$：网络分析计算的最短路径时间
- $F_{detour}$：曲折系数（由巷道密度、障碍物密度决定）
- $T_{wait}$：排队时间（由设施容量/需求比决定）
- $T_{service}$：办理时间（由设施类型决定）

**曲折系数计算**：
$$F_{detour} = 1 + \alpha \times (1 - D_{road\_density} / D_{max})$$

其中：
- $\alpha$：障碍系数（城中村取1.2，高端社区取1.0）
- $D_{road\_density}$：路网密度
- $D_{max}$：研究区最大路网密度

### 4.2 时间窗口匹配度

**定义**：设施开放时间与居民可出行时间的匹配程度

$$M_{time} = \frac{T_{overlap}}{T_{available}}$$

**时段划分**：
- **白天**：08:00-18:00（标准工作时段）
- **夜间**：18:00-22:00（下班后时段）
- **深夜**：22:00-08:00（应急时段）

### 4.3 社区类型分类
| 社区类型 | 识别指标 | 预期时间贫困特征 |
|---------|---------|----------------|
| 城中村 | 建筑密度高、路网稀疏、无封闭管理 | 高曲折系数、高排队、开放时间不友好 |
| 老旧小区 | 建于1990-2000年、设施老化 | 中等时间成本 |
| 普通商品房 | 2000年后建成、封闭管理 | 较低时间成本 |
| 高端社区 | 容积率低、物业完善 | 最低时间成本 |

### 4.4 时间贫困指数构建

**分项指数**：
1. **移动时间指数**（MTI）：$T_{travel} / T_{threshold}$
2. **等待时间指数**（WTI）：$T_{wait} / T_{threshold}$
3. **窗口匹配指数**（TMI）：$1 - M_{time}$
4. **综合时间贫困指数**（CTPI）：
$$CTPI = w_1 \times MTI + w_2 \times WTI + w_3 \times TMI$$

其中权重可由熵权法或层次分析法确定。

## 五、数据处理流程

### 5.1 坐标系统一
```
原始数据（GCJ-02火星坐标）
    ↓ 纠偏算法（偏移量约540m经度/335m纬度）
WGS84（EPSG:4326）
    ↓ 投影变换
CGCS2000 3度带（EPSG:4527，中央经线114°E）
```

### 5.2 设施分类体系
```
POI 10大类 → 3大服务类型 → 8小类

餐饮购物类：
  - 便利店（重点考察夜间服务）
  - 超市
  - 药店

公共服务类：
  - 医院/诊所
  - 学校/教育机构
  - 银行

生活服务类：
  - 快递站点
  - 菜市场
  - 社区服务中心
```

### 5.3 小区AOI匹配
1. 叠加小区AOI与POI数据
2. 统计每个小区的设施类型数量
3. 识别小区边界内的设施开放时间分布
4. 标注社区类型标签

## 六、研究区域：深圳市南山区

### 6.1 选择理由
- 数据完整：有现成的POI、道路、建筑数据
- 空间异质性强：包含城中村（白石洲）、老旧小区、普通商品房、高端社区（深圳湾）
- 昼夜人口活动差异显著：科技园白天高密度、夜间低密度

### 6.2 典型案例对比
| 社区 | 位置 | 特征 | 预期时间成本 |
|------|------|------|-------------|
| 白石洲 | 南山区北部 | 城中村，高密度 | 高 |
| 蔚蓝海岸 | 后海 | 普通商品房 | 中 |
| 深圳湾社区 | 深圳湾 | 高端社区 | 低 |
