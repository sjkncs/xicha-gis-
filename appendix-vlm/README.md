# Appendix: VLM-annotated Street View Images

论文附录图片——VLM（视觉语言模型）标注的街景图像，包含障碍物分类、可见性分析和空间感知评估。

## 目录说明

```
appendix-vlm/
├── appendix_annotated/   # VLM标注后的附录图片
│   ├── Village/         # 村庄/城中村场景
│   ├── 光明区/           # 光明区场景
│   ├── 南山区/           # 南山区场景
│   ├── 宝安区/           # 宝安区场景
│   └── 龙岗区/           # 龙岗区场景
└── appendix_raw/       # 原始街景图片（标注前）
    ├── Village/
    ├── 光明区/
    ├── 南山区/
    ├── 宝安区/
    └── 龙岗区/
```

## 图片命名规则

- 格式：`{经度}_{纬度}_{方向}_{年份}_annot.{ext}`
- 方向：N (北)、S (南)、E (东)、W (西)
- `_annot` 后缀表示已VLM标注

## 标注内容

VLM对每张街景图像进行了以下分析：
- 障碍物类型识别（建筑物、树木、电线杆、围墙等）
- 障碍物高度估计
- 视线遮挡程度评估
- 街道空间开阔度评分
